# OpenBrep v0.5.3 发布说明

发布日期：2026-03-03

## 本版定位

v0.5.3 聚焦两件事：

1. **知识库升级**：整合拆书增量内容，增强 create/debug 检索命中质量与示例可用性。
2. **品牌可见性增强**：README 顶部加入项目 logo，提升入口识别度。

## 主要变化

### 1) 知识库增量整合（Pro）

- 新增/增强 Pro 文档：
  - `GDL_10_3D_Commands_Full`（补充 RULED/COONS/MASS/TUBEA 与布尔组命令、PRISM 变体）
  - `GDL_09_Error_Dictionary`（追加结构化错误模式）
  - `GDL_11_2D_Advanced`（新增，2D 高级命令）
  - `GDL_12_Examples_Verified`（新增，可编译 few-shot 示例）

### 2) 新增命令索引

- 新增 `knowledge/GDL_command_index.md`（Free 层）
- 自动汇总命令与所属 Pro 文档映射，为精准路由提供索引基础。

### 3) 检索路由与评分升级

- `openbrep/knowledge.py`：
  - `_layers` 增加新 Pro 文档加载策略（create/debug）
  - 新增 `_build_command_index()`，读取 `GDL_command_index.md`
  - `get_relevant()` 增加**命令级精确匹配加权**（如 `PRISM_`、`COONS`）
  - `get_by_task_type()` 增加 token 预算保护：超预算时优先保留 Pro 层

### 4) README 品牌增强

- `README.md` 顶部新增 logo 展示（`assets/logo.png`）
- `assets/` 纳入版本管理，便于跨环境一致展示

## 版本号同步

- 代码版本更新至 `0.5.3`：
  - `pyproject.toml`
  - `openbrep/__init__.py`
- 手册版本更新：
  - `docs/manual.md`
- README/README.zh-CN 版本历史补充 `v0.5.3` 条目。

## 与上一版关系

相较 v0.5.2，v0.5.3 的重点从“版本标注规范化”转向“知识能力增强 + 文档入口优化”，不改变核心运行流程。
